基本信息

   项目介绍

项目结构：

        ├─permissions      管理后台
        │    ├─db  数据库SQL脚本
        │    │
        │    ├─modules  模块
        │    │    ├─job 定时任务
        │    │    └─sys 系统管理(核心)
        │    │
        │    └─resources
        │        ├─statics  静态资源
        │        ├─template 系统页面
        │        │    ├─modules      模块页面
        │        │    ├─login.html   登录页面
        │        │    ├─index.html   主页
        │        └─application.yml   全局配置文件
        │        └─application.yml   数据源配置文件
  • 采用Spring、MyBatis、Shiro框架，开发的一套通用权限管理方案

技术选型：

  • 核心框架：Spring Boot 2.0
  • 安全框架：Apache Shiro 1.4
  • 视图框架：Spring MVC 5.0
  • 持久层框架：MyBatis 3.3
  • 定时器：Quartz 2.3
  • 数据库连接池：Druid 1.1
  • 日志管理：SLF4J 1.7、Log4j
  • 页面交互：Vue2.x

软件需求：

  • jdk1.8
  • MySQL5.7
  • Maven3.5
  • IntelliJ IDEA
